# Google Search Console 設定ガイド

## Search Consoleとは

Google Search Consoleは、Googleの検索結果でのサイトのパフォーマンスを監視・管理できる無料ツールです。

### できること：
- 検索キーワードの分析（どんなキーワードで検索されているか）
- クリック数・表示回数・CTR・掲載順位の確認
- インデックス状況の確認（Googleに認識されているページ数）
- サイトマップの送信（新しい記事を早くインデックスさせる）
- モバイルユーザビリティの問題検出
- セキュリティ問題の通知

---

## 設定手順

### ステップ1: Search Consoleにアクセス

1. https://search.google.com/search-console にアクセス
2. Googleアカウントでログイン（Google Analyticsと同じアカウント推奨）
3. 「今すぐ開始」をクリック

### ステップ2: プロパティを追加

2つの方法がありますが、**URLプレフィックス**を選択してください：

- **URLプレフィックス**: `https://k005.net` と入力
  - ※ドメインプロパティよりも設定が簡単です

「続行」をクリック

### ステップ3: 所有権の確認

以下の方法から1つを選択します：

#### 【推奨】HTMLタグ（最も簡単）

1. 「HTMLタグ」を選択
2. 表示されるメタタグをコピー：
   ```html
   <meta name="google-site-verification" content="XXXXXXXXXXXXXXXXXXXX" />
   ```
3. このタグを以下のいずれかの方法でサイトに追加：

**方法A: Insert Headers and Footersプラグインを使用（Google Analyticsと同じ）**
- 設定 > Insert Headers and Footers
- 「Scripts in Header」にメタタグを貼り付け
- 保存

**方法B: テーマヘッダーに直接追加**
- 外観 > テーマファイルエディター > header.php
- `<head>` タグの直後に貼り付け
- 保存

4. Search Console画面に戻って「確認」をクリック

#### 【または】Googleアナリティクス（すでにGA4を設定している場合）

1. 「Googleアナリティクス」を選択
2. GA4測定ID（G-FXW5P6VDSJ）が検出されれば自動的に確認完了
3. 「確認」をクリック

※ただし、この方法はGA4のコードがすでにサイトに設置されている必要があります

---

### ステップ4: サイトマップの送信（重要！）

所有権確認後、すぐにサイトマップを送信しましょう。

1. Search Console左メニューの「サイトマップ」をクリック
2. 「新しいサイトマップの追加」に以下を入力：
   ```
   sitemap.xml
   ```
   または
   ```
   wp-sitemap.xml
   ```
   （WordPressは自動でサイトマップを生成します）

3. 「送信」をクリック

4. ステータスが「成功しました」になればOK

## 確認後にできること

### 1. 検索パフォーマンスの確認
- 左メニュー「検索パフォーマンス」
- どんなキーワードで検索されているか
- 各記事のクリック数・表示回数

### 2. URL検査
- 個別のページがGoogleにインデックスされているか確認
- 新しい記事を公開したら「インデックス登録をリクエスト」で早く検索結果に表示

### 3. カバレッジ
- インデックスされているページ数
- エラーがあるページの確認

## トラブルシューティング

### 「確認できませんでした」と表示される場合

1. **HTMLタグが正しく設置されているか確認**
   - https://k005.net にアクセス
   - 右クリック > ページのソースを表示
   - `google-site-verification` で検索
   - メタタグが `<head>` 内に表示されていればOK

2. **数分待ってから再試行**
   - キャッシュの関係で反映に時間がかかる場合があります

3. **別の確認方法を試す**
   - HTMLファイルのアップロード
   - DNS レコード（高度な方法）

## 次のステップ

1. **まずGoogle Analyticsのコードを設置**
   - Insert Headers and Footersプラグインで設置済みならそのまま使える

2. **Search Consoleの設定**
   - 同じプラグインでHTMLタグも追加
   - またはGoogleアナリティクスで自動確認

3. **サイトマップ送信**
   - `wp-sitemap.xml` を送信

4. **定期的に確認**
   - 週1回程度、検索パフォーマンスをチェック
   - 新記事公開時にURL検査 > インデックス登録リクエスト
