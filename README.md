# パワースポット記事自動生成システム

このプロジェクトは、SEO最適化されたパワースポット記事を生成し、WordPressに投稿するための統合システムです。

## 📁 プロジェクト構成

```
powerspot-content-generator/
├── instructions/              # 記事生成インストラクション
│   ├── ARTICLE_GENERATION_MASTER.md  # ★メインインストラクション（これを読み込むだけでOK）
│   ├── DECISIONS.md          # 設計判断・重要な決定事項
│   ├── writing-guidelines.md # 文章スタイルガイド
│   ├── quality-checklist.md  # 品質チェックリスト
│   └── example-high-quality.md # 高品質記事の例
├── articles/                  # 生成された記事
│   └── izumo-taisha.md       # 出雲大社の記事（サンプル）
├── post-from-markdown-styled.js  # WordPress投稿スクリプト
├── generate-html-preview.js   # HTMLプレビュー生成スクリプト
├── powerspot_article_prompt_v2.md # プロンプト（参考資料）
├── isejingu_article_v2_revised.md # 伊勢神宮記事（参考資料）
├── .env                       # 環境変数（WordPress認証情報）
└── README.md                  # このファイル
```

## 🚀 使い方

### 1. 記事生成

Claude Codeで以下のように指示：

```
instructions/ARTICLE_GENERATION_MASTER.md の指示に完全に従って、
【伏見稲荷大社】のパワースポット記事を作成してください。

【基本情報】
- スポット名: 伏見稲荷大社
- 所在地: 京都府京都市伏見区深草薮之内町68
- 主祭神: 宇迦之御魂大神
- 主なご利益: 商売繁盛、五穀豊穣
- 特徴: 千本鳥居が有名

必ず以下を守ってください：
- 文字数: 4,500-5,000字
- 五行理論には一切触れない
- すべての必須セクションを含める
- 具体的な数字・時刻を豊富に
```

### 2. HTMLプレビュー生成

```bash
node generate-html-preview.js articles/izumo-taisha.md
```

ブラウザで `articles/izumo-taisha-preview.html` を開いて確認

### 3. WordPressに投稿

```bash
node post-from-markdown-styled.js articles/izumo-taisha.md
```

下書きとして投稿されます。WordPress管理画面で確認・公開してください。

## 📋 記事の仕様

### 文字数
**4,500-5,000文字（厳守）**

### 必須セクション（順序も重要）
1. タイトル
2. 導入部（300-400字）
3. このスポットの魅力（800-1,000字）
4. ご利益・期待できる効果（500-700字）
5. ベストな訪問時期（600-800字）
6. 参拝・見学ガイド（700-900字）
7. **スポットの基本情報（300-400字）** ← ここに配置（2024年11月更新）
8. 周辺情報（800-1,000字）
9. 訪問者の口コミ・体験談（300-400字）
10. よくある質問（200-300字）
11. まとめ（400-500字）

### ❌ 絶対に書いてはいけないこと
- 五行理論の説明（木火土金水）
- 12位置システムの言及
- 60分類・縁タイプの詳細
- 「○○タイプの人におすすめ」
- 生年月日や名前との相性分析
- バイオリズムとの関連
- 統計学的根拠・計算式

これらは診断アプリの価値を下げるため、記事では一切触れません。

## 🎯 ビジネスモデル

```
SEO集客（パワースポット記事）
  ↓
無料診断アプリ（縁プロファイル診断）
  ↓
有料詳細レポート（¥2,980）
```

記事の役割：
1. SEO集客（「[スポット名] パワースポット」で上位表示）
2. 価値提供（他サイト以上の充実した実用情報）
3. 自然な診断誘導（記事末尾の控えめなリンク）

## 📊 品質基準

記事は以下の基準を満たす必要があります：

- ✅ 文字数：4,500-5,000字
- ✅ 具体的な数字：15箇所以上
- ✅ 五感描写：3箇所以上
- ✅ 体験談・エピソード：2-3箇所
- ✅ 周辺グルメ：3-5店舗
- ✅ モデルコース：半日＋1日
- ✅ FAQ：3-5個
- ✅ コンテンツ独自性：既存サイトのコピーでないこと

詳細は `instructions/quality-checklist.md` を参照

## 🎨 スタイル仕様

### HTML生成の特徴
- セクションごとに色分け・スタイリング
- 基本情報・アクセスは青系ボックス
- 口コミは引用スタイル
- FAQは絵文字付き（❓💡）
- 季節のおすすめ度は★表示

### CTA（診断誘導）の方針
記事末尾に**控えめなリンク**のみ配置：
```
💡 あなたに最適なパワースポットを知りたい方は
無料の相性診断をお試しください（3分で完了）
```

派手なCTA・有料商品の直接言及は避ける。サイト全体（ヘッダー・サイドバー）で診断を訴求する戦略。

## 📝 重要な設計判断

### セクション順序の変更（2024年11月17日）

**理由：** 読者は興味を持ってから実用情報を見る

- **変更前：** 導入 → 基本情報（アクセス・営業時間）→ 魅力 → ...
- **変更後：** 導入 → 魅力 → ご利益 → 訪問時期 → 参拝ガイド → **基本情報** → 周辺情報 → ...

読者の関心が高まってから、アクセスや営業時間などの実用情報を提示することで、離脱を防ぐ。

### CTAの最小化（2024年11月17日）

**理由：** 記事と診断の間に文脈のつながりがない

当初は記事末尾に「診断を始める」ボタン付きの大きなCTAセクションがあったが、パワースポット記事から突然診断・有料商品に誘導するのは不自然。

**解決策：**
- 記事末尾は控えめなテキストリンクのみ
- 診断訴求はサイト全体（ヘッダー・サイドバー・固定バナー）で行う
- 記事の役割は「価値提供＋SEO集客」に専念

詳細は `instructions/DECISIONS.md` を参照
