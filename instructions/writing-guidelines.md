# パワースポット記事作成ガイドライン

## 📌 記事の目的

- **ターゲット読者**: 30-60代、スピリチュアルや開運に興味がある人
- **記事の役割**: 実際に訪問したくなる、具体的で実用的な情報を提供
- **最終目標**: 記事を読んで「行ってみたい」と思わせ、縁診断への導線を作る

---

## ✍️ 文体とトーン

### 基本方針
- **一人称**: 「私」を使った体験談風（過度に使わない）
- **語尾**: 断定的に書く（「〜です」「〜でした」）
- **避けるべき表現**:
  - ❌ 「〜と言われています」→ ✅ 「〜です」
  - ❌ 「〜かもしれません」→ ✅ 「〜でしょう」
  - ❌ 「〜という話があります」→ ✅ 「〜という伝承があります」

### トーンの例
```
❌ 悪い例: 「伊勢神宮は神聖な場所だと言われています」
✅ 良い例: 「伊勢神宮の宇治橋を渡った瞬間、空気が一変します。背筋が伸びるような清々しさと、心地よい緊張感。これが、2000年の歴史が紡ぐ神聖なエネルギーです」
```

## 🎨 具体性を高める技法

### 1. 五感を使った描写

```
❌ 「美しい景色が広がっています」
✅ 「早朝5時、朝霧に包まれた境内。鳥居の向こうから差し込む朝日が、樹齢300年の杉並木を黄金色に染め上げます。足元の玉砂利を踏む音だけが静寂を破り、どこからか梅の香りが漂ってきます」

視覚 + 聴覚 + 嗅覚 = 臨場感
```

### 2. 具体的な数字・時刻

```
❌ 「早朝がおすすめ」
✅ 「午前5時〜7時が最も空いており、神聖な雰囲気を独り占めできます」

❌ 「近い」
✅ 「駅から徒歩12分、緩やかな上り坂を進みます」
```

### 3. 比較・対比

```
❌ 「パワーが強い」
✅ 「出雲大社の包み込むような優しいエネルギーとは対照的に、ここは心身を浄化する鋭いエネルギーを感じます」
```

### 4. ストーリー・エピソード

```
❌ 「ご利益があります」
✅ 「訪問の3ヶ月後、10年探し続けた理想の仕事に出会えました。偶然かもしれませんが、参拝時に感じた『背中を押される』ような感覚は今も忘れられません」
```

## 🌟 独自性を出す方法

### 地元民しか知らない情報
- 裏参道・隠れスポット
- 地元の人がよく訪れる時間帯
- 駐車場の混雑を避けるコツ
- 近くの穴場グルメ

### 失敗談・注意点
```
良い例:
「夏の午後2時に訪れた際、石段の照り返しで熱中症寸前に。水分は必須です。木陰のベンチで休憩しながら、ゆっくり参拝することをおすすめします」
```

### 季節・時間帯による違い
```
良い例:
「春の早朝は桜と朝霧のコラボレーション。秋の夕暮れは紅葉が夕日に照らされて幻想的。冬の雪化粧した境内は静謐そのもの。同じ場所でも、訪れる時期で全く違う表情を見せます」
```

## 📍 各セクションの書き方

### パワースポットとしての特徴（最重要）

このセクションが記事の核心。以下を必ず含める：

1. **第一印象**（到着時の感覚）
   - 「鳥居をくぐった瞬間〜」
   - 「参道を歩き始めると〜」

2. **エネルギーの質**
   - 浄化系 / 活性化系 / 癒し系
   - どの部分で最も強く感じるか

3. **他との違い**
   - 「伊勢神宮の荘厳さとは違い〜」
   - 「都会の神社にはない〜」

4. **具体的な体験談**
   - 実際に訪れた時の出来事
   - 参拝後の変化

**最低500文字、理想は700文字**

### 見どころ

単なるリストではなく、各スポットを描写：

```
❌ 悪い例:
「見どころ:
- 本殿
- 奥社
- 滝」

✅ 良い例:
### 本殿
朱塗りの本殿は、樹齢500年の杉木立に囲まれています。正午、木漏れ日が社殿を照らす瞬間は息を呑む美しさ。参拝客が途切れる14時頃が狙い目です。

### 奥社（最強パワースポット）
本殿から徒歩15分、石段を200段登った先にある奥社。ここが真の聖域です。周囲を巨木に囲まれた小さな社殿ですが、空気の澄み方が別次元。深呼吸すると、心の奥底まで浄化される感覚があります。

### 霊験の滝
奥社の脇道を進むと、高さ15mの滝が現れます。滝壺の近くに立つと、マイナスイオンを全身で感じられます。滝の音以外何も聞こえない、瞑想に最適な場所です。
```

## 🚫 絶対に避けるべきこと

1. **ありきたりな表現**
   - 「パワーをもらえる」
   - 「神秘的な雰囲気」
   - 「心が洗われる」

   → 具体的にどんなパワー・雰囲気・感覚かを描写する

2. **情報の羅列**
   - 箇条書きだけで終わらない
   - 各項目に説明を加える

3. **曖昧な情報**
   - 「近くにある」→ 「徒歩5分」
   - 「たくさんある」→ 「50台」
   - 「おすすめ」→ 「なぜおすすめか」を説明

4. **事実誤認**
   - 不確かな情報は書かない
   - 伝承は「〜という伝承があります」と明記

## 📏 品質チェックリスト

記事完成後、以下を確認：

- [ ] 文字数が2,500文字以上ある

- [ ] 具体的な数字・時刻が5箇所以上ある
- [ ] 五感を使った描写が3箇所以上ある
- [ ] 独自の視点・情報が含まれている
- [ ] 「〜と言われています」などの曖昧表現がない
- [ ] 実際に訪れたかのような臨場感がある
- [ ] 写真を撮りたくなるスポットが紹介されている
- [ ] 失敗を避けるための注意点が含まれている
- [ ] 最後に縁診断への自然な導線がある

## 💎 高品質記事の黄金律

1. **具体的であれ**: 抽象的な表現は徹底的に排除
2. **独自性を出せ**: 他のサイトにない情報を必ず含める
3. **実用的であれ**: 読者が実際に訪問する時に役立つ情報
4. **臨場感を出せ**: 読んだだけで訪れた気分になれる描写
5. **ストーリーを語れ**: 情報の羅列ではなく、物語として
