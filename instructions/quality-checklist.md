# 記事品質チェックリスト

記事を公開する前に、このチェックリストで品質を確認してください。

---

## 📋 基本要件

### Front Matter（メタデータ）
- [ ] `title`が設定されている（50文字以内）
- [ ] `excerpt`が設定されている（120文字以内）
- [ ] タイトルにスポット名とキーワードが含まれている
- [ ] excerptが魅力的で読みたくなる内容

### 構成
- [ ] すべての必須セクションが揃っている
  - [ ] 基本情報
  - [ ] パワースポットとしての特徴
  - [ ] ご利益
  - [ ] 見どころ
  - [ ] アクセス方法
  - [ ] 訪問のベストタイミング
  - [ ] 周辺情報
  - [ ] まとめ

### 文字数
- [ ] 総文字数が2,500文字以上
- [ ] 「パワースポットとしての特徴」が500文字以上
- [ ] 「見どころ」が500文字以上

---

## ✍️ 文章品質

### 具体性
- [ ] 具体的な数字が5箇所以上ある
  - 例: 「徒歩12分」「樹齢300年」「午前5時」「石段200段」
- [ ] 具体的な時刻が3箇所以上ある
  - 例: 「5時」「14時30分」「17時頃」
- [ ] 具体的な料金・台数が明記されている
  - 例: 「無料」「500円」「駐車場50台」

### 五感描写
- [ ] 視覚的描写が3箇所以上
  - 例: 「朝日が杉並木を黄金色に染め」
- [ ] 聴覚的描写が1箇所以上
  - 例: 「玉砂利を踏む音」「せせらぎの音」
- [ ] その他の感覚描写がある（嗅覚・触覚など）
  - 例: 「梅の香り」「ひんやりとした空気」

### 臨場感
- [ ] 「実際に訪れたかのような」具体的な体験描写がある
- [ ] 季節・時間帯による違いが説明されている
- [ ] 第一印象の描写がある（到着時・鳥居をくぐった瞬間など）

---

## 🌟 独自性・差別化

### オリジナル情報
- [ ] ガイドブックにない情報が1つ以上ある
  - 地元民しか知らない情報
  - 裏道・隠れスポット
  - 混雑を避けるコツ
- [ ] 失敗談・注意点が含まれている
  - 例: 「夏の午後は避けるべき」「駐車場は8時までに」
- [ ] 他のパワースポットとの比較がある
  - 例: 「伊勢神宮の荘厳さとは対照的に〜」

### ストーリー性
- [ ] エピソード・体験談が含まれている
- [ ] 情報の羅列ではなく、読み物として成立している
- [ ] 読者が「行ってみたい」と思える内容

---

## 🚫 避けるべき表現チェック

以下の曖昧な表現が使われていないか確認：

- [ ] 「〜と言われています」がない
- [ ] 「〜かもしれません」がない
- [ ] 「〜という話があります」がない
- [ ] 「パワーがある」だけで終わっていない
- [ ] 「美しい」だけで終わっていない
- [ ] 「神秘的」だけで終わっていない

すべて具体的な描写に置き換えられているか？

---

## 🎯 SEO最適化

### キーワード
- [ ] タイトルに主要キーワードが含まれている
  - スポット名
  - 「パワースポット」
  - 「ご利益」または「アクセス」
- [ ] 見出し（##）に自然にキーワードが含まれている
- [ ] 地域名が適切に使われている
- [ ] キーワードの詰め込み過ぎがない（自然な文章）

---

## 📍 実用性

### アクセス情報
- [ ] 最寄り駅からの具体的なルートが書かれている
- [ ] 徒歩の所要時間が明記されている
- [ ] 車でのアクセス（IC名）が書かれている
- [ ] 駐車場情報（台数・料金）が書かれている
- [ ] 混雑時の対策が含まれている

### タイミング情報
- [ ] おすすめの時期が書かれている（理由付き）
- [ ] おすすめの時間帯が書かれている（理由付き）
- [ ] 避けるべき時期・時間帯が書かれている
- [ ] 滞在時間の目安が書かれている

### 周辺情報
- [ ] 近隣の他のスポットが紹介されている
- [ ] 地元グルメ・名物が紹介されている
- [ ] 宿泊施設の情報がある（該当する場合）

---

## 💎 読者体験

### 読みやすさ
- [ ] 段落が適切に分かれている（長すぎない）
- [ ] 見出しが適切に使われている
- [ ] 箇条書きと文章のバランスが良い
- [ ] リズムよく読める

### 魅力
- [ ] 冒頭から引き込まれる内容
- [ ] 「行ってみたい」と思わせる描写
- [ ] SNSでシェアしたくなる情報
- [ ] 写真を撮りたくなるスポットの紹介

### CTA（行動喚起）
- [ ] まとめに縁診断への導線がある
- [ ] 「五行思想に基づく縁診断で〜」の文言がある
- [ ] 自然な流れで導線が設置されている（唐突でない）

---

## 🔍 最終チェック

### 事実確認
- [ ] スポット名・住所・祭神などの基本情報が正確
- [ ] アクセス情報が最新（駅名・路線名など）
- [ ] 料金・営業時間が正確（または「要確認」と記載）
- [ ] 不確かな情報は「〜という伝承」と明記

### 表記統一
- [ ] 数字の表記が統一されている（全角/半角）
- [ ] スポット名の表記が一貫している
- [ ] 「〜です」「〜ます」調で統一されている

### 誤字脱字
- [ ] 誤字脱字がない
- [ ] 変換ミスがない
- [ ] 句読点が適切

## 📝 改善のヒント

### 得点が低い項目別の対処法

**具体性が低い場合:**
- 曖昧な表現を探して、具体的な数字・時刻に置き換える
- 「徒歩すぐ」→「徒歩3分」
- 「早朝」→「午前5時〜7時」

**独自性が低い場合:**
- Google検索で上位記事を確認し、それにない情報を追加
- 地元民の視点を加える
- 失敗談・注意点を追加

**臨場感が低い場合:**
- 五感描写を追加（色・音・香り・感触）
- 季節・時間帯による違いを詳しく
- 「〜です」→「〜でした（体験談）」に変更

**実用性が低い場合:**
- アクセスセクションを詳しく
- Googleマップで実際のルートを確認
- 駐車場・所要時間など数字を追加

**読みやすさ・魅力が低い場合:**
- 長い段落を分割
- 箇条書きを追加
- 冒頭の第一印象を強化
