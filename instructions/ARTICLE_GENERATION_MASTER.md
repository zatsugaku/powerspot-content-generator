# パワースポット記事作成マスターインストラクション

**このファイルを読み込むだけで、高品質なパワースポット記事を作成できます**

---

## 🎯 記事作成の目的

### ビジネスモデル
```
SEO集客（パワースポット記事）
  ↓
無料診断アプリ（縁プロファイル診断）
  ↓
有料詳細レポート（¥2,980）
```

### この記事の役割
1. **SEO集客**: 「[スポット名] パワースポット」で上位表示
2. **価値提供**: 他サイト以上の充実した実用情報
3. **興味喚起**: パワースポットの魅力を最大限に伝える
4. **疑問創出**: 「自分に合うのかな？」という疑問を持たせる
5. **診断誘導**: 自然に無料診断へ誘導

---

## ⚠️ 絶対に守るべきルール

### ❌ 絶対に書いてはいけないこと
- 五行理論の説明（木火土金水）
- 12位置システムの言及
- 60分類・縁タイプの詳細
- 「○○タイプの人におすすめ」
- 生年月日や名前との相性分析
- バイオリズムとの関連
- 統計学的根拠・計算式

### 理由
1. 読者の目的とズレる（スポット情報が知りたいだけ）
2. 専門的すぎて離脱される
3. 診断アプリの価値を下げる
4. 怪しいサイトと思われるリスク

## 📏 記事の基本仕様

### 文字数
**4,500-5,000文字（厳守）**

### 文体
- 丁寧語（です・ます調）
- 親しみやすく、読者に寄り添う
- 具体的で臨場感がある
- 中立的（特定の宗教を優遇・批判しない）

### 避けるべき表現
❌ 「絶対に効果がある」等の断定
❌ 「五行相生」「天干地支」等の専門用語
❌ 「○○タイプの人におすすめ」等の相性判断
❌ 「スピリチュアル」「波動」等の曖昧な表現
❌ 科学的根拠のない効能の断言

### 推奨する表現
✅ 「多くの方が～を感じたと語っています」（体験ベース）
✅ 「古くから～のご利益があるとされています」（伝統ベース）
✅ 「この場所には特別な雰囲気があります」（感覚的）
✅ 「訪れた後、前向きな気持ちになれた」（具体的体験）

## 📝 記事構成テンプレート（必須セクション）

### 1. タイトル
**形式**: [スポット名] | [都道府県]のパワースポット完全ガイド【アクセス・ご利益・周辺情報】

**例**:
- 伊勢神宮（内宮）| 三重県のパワースポット完全ガイド【参拝方法・周辺グルメ】
- 出雲大社 | 島根県のパワースポット完全ガイド【縁結び・アクセス情報】

### 2. 導入部（300-400字）
- スポットの第一印象と特別な魅力
- なぜこの場所が特別なのか
- 読者の興味を引く印象的なエピソード
- どんな体験ができるのか

### 3. このスポットの魅力（800-1,000字）

#### 歴史と由来
- 創建年代・由緒
- 祀られている神様や仏様
- 歴史的エピソード・伝説
- 文化財指定状況
- なぜこの場所が選ばれたのか

#### 建築・自然の特徴
- 境内・建物の見どころ
- 自然環境の特徴（森・川・山・海など）
- 季節ごとの景観
- フォトスポット

#### このスポットの特別なポイント
- 他のスポットにはない独自性
- 地元での位置づけ
- 有名人の訪問歴
- メディア掲載歴

### 4. ご利益・期待できる効果（500-700字）

#### 一般的に知られているご利益
- 恋愛成就・縁結び
- 金運・商売繁盛
- 健康長寿・病気平癒
- 学業成就・合格祈願
- 厄除け・開運
- その他（スポット固有のご利益）

#### このスポットの特徴的なパワー
- 「新しい始まり」のエネルギー
- 「浄化と癒し」の力
- 「決断力」を高める
- 「人間関係」を円滑にする
※抽象的な表現で、具体的な相性判断は避ける

#### 訪れた人の声
- 「気持ちが前向きになった」
- 「心が洗われる感じがした」
- 「不思議と決断できた」
※体験談として、効果を断定しない表現

### 5. ベストな訪問時期（600-800字）

#### 季節別の魅力（春夏秋冬各150-200字）
- 見どころ
- 気候・混雑状況
- この時期ならではの魅力
- おすすめ度: ★★★★★

#### 時間帯別のおすすめ
- **早朝（開門直後）**: 人が少ない・空気が清々しい
- **午前中**: 観光に最適
- **午後**: 光の入り方が美しい（など）
- **夕方**: 夕日が美しい（該当する場合）

#### 混雑回避のコツ
- 平日vs週末
- 避けるべき時期（正月・GWなど）
- 穴場の時間帯

### 6. 参拝・見学ガイド（700-900字）

#### 基本的な参拝作法
1. **鳥居のくぐり方**: 一礼してから・真ん中を避ける
2. **手水舎での清め方**: 正しい手順を詳しく説明
3. **参道の歩き方**: 端を歩く・真ん中は神様の通り道
4. **拝殿での参拝**:
   - お賽銭の入れ方
   - 二拝二拍手一拝（神社の場合）
   - 合掌（お寺の場合）
   - 祈り方のポイント

#### このスポット特有のポイント
- 特別な参拝ルート
- 見逃せない場所（本殿以外）
- パワーが強いと言われる場所
- お守り・御朱印情報
- 撮影可能場所・禁止場所

#### 服装・マナー
- 推奨される服装
- 避けるべき服装
- 持参すると便利なもの
- 禁止事項・注意事項

### 7. スポットの基本情報（300-400字）

#### アクセス
- **所在地**: [完全な住所]
- **最寄駅**: [駅名・路線名]
- **駅からのアクセス**:
  - 徒歩: ○分（具体的なルート）
  - バス: ○○バス「○○」行き、「○○」下車（所要時間・料金）
  - タクシー: 所要時間・おおよその料金
- **車でのアクセス**:
  - [IC名]より○分
  - カーナビ設定用情報

#### 基本情報
- **参拝/見学時間**: [詳細な時間・季節変動]
- **定休日**: [あれば]
- **拝観料/入場料**: [料金詳細]
- **所要時間**: 標準○分～じっくり○時間
- **駐車場**: 有無・台数・料金・混雑状況
- **公式サイト**: [URL]
- **お問い合わせ**: [電話番号]

### 8. 周辺情報（800-1,000字）

#### 周辺の観光スポット
- **徒歩圏内**（○分）: [スポット名と簡単な説明]
- **車で○分**: [スポット名と簡単な説明]
- **セットで回りたい**: おすすめの周遊ルート

#### おすすめグルメ・カフェ（3-5店舗）
各店について：
- ジャンル・特徴
- 名物メニュー
- 予算感
- スポットからの距離・アクセス
- 営業時間・定休日

#### お土産・特産品
- 定番のお土産
- 地元の名物
- 購入できる場所

#### 宿泊施設
- 近隣の主要ホテル・旅館
- 予算別のおすすめ
- 温泉地が近い場合は言及

#### モデルコース
**半日コース**と**1日コース**を具体的な時間で提示

### 9. 訪問者の口コミ・体験談（300-400字）

#### 肯定的な声（3-4人分）
- 年齢・性別・地域を明記
- 具体的な体験内容
- リアリティのある内容

#### 参考になる情報
- 混雑状況の実際
- 所要時間の実感
- 意外な発見

### 10. よくある質問（200-300字）

**3-5個のQ&A**
- 所要時間
- 駐車場混雑
- 御朱印
- ペット同伴
- 車椅子対応
- 外宮・内宮の順序（該当する場合）
- お賽銭の金額
- 服装

### 11. まとめ＋診断誘導CTA（400-500字）

#### スポットの核心的価値
- このスポットを一言でまとめる
- 訪れるべき理由
- どんな体験が得られるか

#### 読者へのメッセージ
- 訪問を後押しする言葉
- 実際に行くことの価値

#### 診断誘導CTA（必須）

```markdown
---

日本には数多くのパワースポットがありますが、それぞれに相性があります。たった3分の無料診断であなたに最適なパワースポットがわかります。

→ [無料で縁プロファイル診断を受ける](https://enguide.info)

```

### 12. 関連記事セクション（自動生成）

**注意**: このセクションはWordPress投稿後に `add-related-links.js` スクリプトによって自動的に追加されます。手動で記述する必要はありません。

自動追加される内容：
- ピラーページ（パワースポット完全ガイド）へのリンク
- 同じ地域のパワースポット記事（最大3件）
- 同じご利益のパワースポット記事（最大3件）

スクリプト実行方法：
```bash
# 全記事に関連リンクを追加
node add-related-links.js

# 特定の記事のみに追加
node add-related-links.js --single=ise-jingu

# 実行前の確認（dry-run）
node add-related-links.js --dry-run
```

## ✍️ 文章品質基準

### 具体性を高める技法

#### 1. 五感を使った描写（必須：3箇所以上）
❌ 「美しい景色が広がっています」
✅ 「早朝5時、朝霧に包まれた境内。鳥居の向こうから差し込む朝日が、樹齢300年の杉並木を黄金色に染め上げます。足元の玉砂利を踏む音だけが静寂を破り、どこからか梅の香りが漂ってきます」

**視覚 + 聴覚 + 嗅覚 = 臨場感**

#### 2. 具体的な数字・時刻（必須：15箇所以上）
❌ 「早朝がおすすめ」
✅ 「午前5時〜7時が最も空いており、神聖な雰囲気を独り占めできます」

❌ 「近い」
✅ 「駅から徒歩12分、緩やかな上り坂を進みます」

#### 3. 比較・対比（推奨：1-2箇所）
❌ 「パワーが強い」
✅ 「出雲大社の包み込むような優しいエネルギーとは対照的に、ここは心身を浄化する鋭いエネルギーを感じます」

#### 4. ストーリー・エピソード（必須：2-3箇所）
❌ 「ご利益があります」
✅ 「訪問の3ヶ月後、10年探し続けた理想の仕事に出会えました。偶然かもしれませんが、参拝時に感じた『背中を押される』ような感覚は今も忘れられません」
