# Google Search Console サイトマップ再送信手順

## 手順

### 1. Search Consoleにアクセス
https://search.google.com/search-console

### 2. サイトマップページを開く
左メニューから「サイトマップ」をクリック

### 3. 既存のサイトマップを削除（任意）
- 送信済みサイトマップの一覧が表示されます
- `wp-sitemap.xml` の右側にある「︙」（3点メニュー）をクリック
- 「サイトマップの削除」を選択

### 4. サイトマップを再送信
- 「新しいサイトマップの追加」の入力欄に以下を入力：
  ```
  wp-sitemap.xml
  ```
- 「送信」をクリック

### 5. ステータスを確認
- ステータスが「成功しました」になればOK
- 「取得できませんでした」と表示される場合は、数分待ってから再試行

## 各記事のインデックス登録をリクエスト（推奨）

新しいURLを早くGoogleに認識させるため、各記事のインデックス登録をリクエストします。

### 手順

1. Search Console画面上部の検索ボックスに記事URLを入力：
   - `https://k005.net/nikko-toshogu/`
   - `https://k005.net/mount-aso/`
   - `https://k005.net/izumo-taisha/`
   - `https://k005.net/kotohira-gu/`
   - `https://k005.net/sefa-utaki/`
   - `https://k005.net/fushimi-inari-taisha/`
   - `https://k005.net/ise-jingu/`

2. 「インデックス登録をリクエスト」をクリック

3. 1〜2分待つ

4. 「インデックス登録をリクエスト済み」と表示されればOK

**7記事すべてでこの手順を繰り返します**

## 補足

### サイトマップが見つからない場合

WordPressは自動的にサイトマップを生成します。以下のURLで確認できます：
- https://k005.net/wp-sitemap.xml

ブラウザでこのURLにアクセスして、XMLファイルが表示されればOKです。

### 反映までの時間

- サイトマップの処理: 数時間〜数日
- インデックス登録リクエスト: 数時間〜1日
- 検索結果への表示: 数日〜数週間
