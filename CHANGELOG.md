# 変更履歴

このファイルには、パワースポット記事自動生成システムのすべての重要な変更が記録されています。

## [2.0.0] - 2024-11-17

### 🔄 構造的変更

#### セクション順序の変更
**変更理由**: 読者は興味を持ってから実用情報を見る

- **変更前の順序**:
  1. 導入部
  2. スポットの基本情報（アクセス・営業時間）← ここにあった
  3. このスポットの魅力
  4. ご利益
  5. ...

- **変更後の順序**:
  1. 導入部
  2. このスポットの魅力
  3. ご利益・期待できる効果
  4. ベストな訪問時期
  5. 参拝・見学ガイド
  6. **スポットの基本情報（アクセス・営業時間）** ← ここに移動
  7. 周辺情報
  8. 訪問者の口コミ
  9. よくある質問
  10. まとめ

**影響範囲**:
- `instructions/ARTICLE_GENERATION_MASTER.md` 更新
- `articles/izumo-taisha.md` セクション移動
- 今後生成する全記事

**ユーザーメリット**:
- 離脱率低下（興味が高まってからアクセス情報を見る）
- 読了率向上（魅力を先に伝える）
- コンバージョン改善（診断リンクまで到達する読者増加）

### 🎨 CTA設計の変更

**変更理由**: 記事と診断の間に文脈のつながりがない

- **変更前**: 記事末尾に大きなCTAセクション
  ```markdown
  ## 「出雲大社、行ってみたい！」と思ったあなたへ

  実は、パワースポットには「人それぞれの相性」があることをご存知ですか？

  [500文字以上の診断説明]

  → **無料診断を始める** [ボタン]

  **詳細レポート** ¥2,980
  [レポート内容の詳細説明300文字]
  ```

- **変更後**: 控えめなテキストリンクのみ
  ```markdown
  💡 あなたに最適なパワースポットを知りたい方は
  無料の相性診断をお試しください（3分で完了）
  ```

**問題点（変更前）**:
1. パワースポット記事から突然診断に誘導するのは不自然
2. 有料商品（¥2,980）の言及が早すぎる
3. 記事と診断の間に論理的なつながりがない
4. 読者が求める情報（スポット情報）と提供する情報（診断）にギャップ

**解決策**:
- 記事末尾は控えめなリンクのみ
- 診断訴求はサイト全体（ヘッダー・サイドバー・固定バナー）で行う
- 記事の役割は「価値提供＋SEO集客」に専念

**影響範囲**:
- `post-from-markdown-styled.js` の `addFooterCTA()` 関数更新
- `generate-html-preview.js` の `addFooterCTA()` 関数更新
- `articles/izumo-taisha.md` まとめセクション簡素化

### 📏 文字数範囲の変更

- **変更前**: 3,500-5,000文字
- **変更後**: **4,500-5,000文字（厳守）**

**変更理由**:
- SEO効果の向上（より詳細なコンテンツ）
- ユーザー満足度向上（充実した情報量）
- 競合との差別化（薄いコンテンツとの差別化）

**影響範囲**:
- `instructions/ARTICLE_GENERATION_MASTER.md` 更新
- `powerspot_article_prompt_v2.md` 更新（3箇所）
- 今後生成する全記事

### ✅ コンテンツ独自性確認プロセスの追加

**新規追加**: 記事生成後にGoogle検索でコンテンツ独自性を確認

**確認方法**:
1. 記事から特徴的なフレーズを5-7個抽出
2. Google検索で完全一致検索（""で囲む）
3. 既存コンテンツとの重複がないことを確認

**確認例（出雲大社記事）**:
- ✅ 「大鳥居をくぐった瞬間から感じる神聖な空気、参道を歩くたびに心が洗われていく感覚」
- ✅ 「朝日が松の間から差し込む光景は、まるで神話の世界に迷い込んだような美しさです」
- ✅ 「伊勢神宮の凛とした浄化のパワーとは対照的に、出雲大社は受け入れてくれる包み込んでくれるような温かいエネルギー」

結果: すべてオリジナルコンテンツと確認

## [1.0.0] - 2024-11-16

### 🎉 初版リリース

#### 記事生成システム

**作成ファイル**:
- `instructions/ARTICLE_GENERATION_MASTER.md` - 統合インストラクション
- `instructions/writing-guidelines.md` - 文章スタイルガイド
- `instructions/quality-checklist.md` - 品質チェックリスト（100点満点）
- `instructions/example-high-quality.md` - 出雲大社の高品質例

**記事仕様**:
- 文字数: 3,500-5,000文字
- 必須セクション: 11個
- 禁止事項: 五行理論、相性判断など7項目

#### WordPress投稿機能

**作成ファイル**:
- `post-from-markdown-styled.js` - スタイル付きHTML変換＋WordPress投稿
- `generate-html-preview.js` - ローカルプレビュー生成

**機能**:
- Markdown → スタイル付きHTML変換
- セクションごとの色分け・スタイリング
- WordPress REST API経由で下書き投稿
- 環境変数での認証情報管理

#### サンプル記事

**生成記事**:
- `articles/izumo-taisha.md` - 出雲大社（11,876字の詳細版）

**参考資料**:
- `powerspot_article_prompt_v2.md` - プロンプト
- `isejingu_article_v2_revised.md` - 伊勢神宮の高品質例

#### ビジネスモデル定義

```
SEO集客（パワースポット記事）
  ↓
無料診断アプリ（縁プロファイル診断）
  ↓
有料詳細レポート（¥2,980）
```

**記事の役割**:
1. SEO集客
2. 価値提供
3. 興味喚起
4. 疑問創出
5. 診断誘導

#### 品質基準の確立

**具体性の要件**:
- 具体的な数字: 15箇所以上
- 五感描写: 3箇所以上
- 体験談・エピソード: 2-3箇所
- 周辺グルメ: 3-5店舗
- モデルコース: 半日＋1日
- FAQ: 3-5個

#### 禁止事項の明確化

**絶対に書いてはいけないこと**:
- 五行理論の説明（木火土金水）
- 12位置システムの言及
- 60分類・縁タイプの詳細
- 「○○タイプの人におすすめ」
- 生年月日や名前との相性分析
- バイオリズムとの関連
- 統計学的根拠・計算式

**理由**: 診断アプリの価値を下げるため
